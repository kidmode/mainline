A Unity3D editor plugin for importing and playing general animation format files.
Author: Alexey Nikitin.
Copyright: © Catalyst Apps.
Version: 3.9

How to:
1)	Install it in Unity like any other plugin.
2)  Generate ".gaf" animation from your ".swf" ( For detailed info go to - http://gafmedia.com ).
3)	Copy a folder with your GAF animation files into a unity project directory.
4)	".gaf" file will be processed and animation asset file will be created.
5)	Create empty GameObject and add GAFMovieClip behavior to it.
6)	Drag’n’drop asset file onto GAFMovieClip interface field.
7)	Load animation.
